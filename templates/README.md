# 个人记账模板包

3套专业记账模板，支持 Excel / WPS / Google Sheets / Numbers。

## 文件清单

- `月度预算表.csv` — 12个月独立工作表模板，收入/支出/结余自动计算
- `支出分类追踪.csv` — 6大支出分类（餐饮/住房/交通/购物/娱乐/其他），月度追踪
- `年度汇总报表.csv` — 全年12个月数据汇总，月度对比
- `使用指南.md` — 详细图文教程，5分钟上手

## 导入方式

### Excel
1. 打开Excel → 数据 → 从文本/CSV导入
2. 选择对应CSV文件，编码选UTF-8
3. 导入后可根据需要调整格式和公式

### Google Sheets
1. 新建Google Sheets → 文件 → 导入
2. 上传CSV文件，选择"替换当前工作表"
3. 自动识别分隔符，直接导入

### WPS表格
1. 打开WPS → 数据 → 导入数据
2. 选择CSV文件，编码UTF-8
3. 导入后格式完整保留
