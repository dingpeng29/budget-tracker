# 个人记账模板 — 使用指南

## 快速上手（5分钟）

### 第一步：导入模板
- **Excel**：数据 → 从文本/CSV导入 → 选择CSV文件（UTF-8编码）
- **WPS**：数据 → 导入数据 → 选择CSV文件
- **Google Sheets**：文件 → 导入 → 上传CSV → 替换当前工作表
- **Numbers**：直接拖入CSV文件即可

### 第二步：设置月度预算
1. 打开"月度预算表"
2. 在"月预算"列填入每月的支出预算
3. 建议参考前3个月平均支出上浮10%

### 第三步：开始记账
1. 每天花2分钟记录当日支出
2. 在"支出分类追踪"中填入对应分类的金额
3. 周末花5分钟核对一次

### 第四步：查看报表
1. 月度预算表自动显示每月结余
2. 支出分类追踪饼图展示消费结构
3. 年度汇总报表查看全年趋势

## 记账小技巧

1. **坚持每日记录**：设置手机提醒，每天固定时间记账
2. **大额支出标记**：超过月收入10%的支出单独标注
3. **月底复盘**：对比预算和实际，找出超支分类
4. **设定储蓄目标**：建议月收入的20-30%作为储蓄目标
5. **使用微信/支付宝账单**：每月初导出账单，批量录入

## 图表使用说明

### Excel/WPS 中创建饼图
1. 选中"支出分类追踪"中全年合计行的6个分类数据
2. 插入 → 饼图 → 选择样式
3. 图表自动展示各分类占比

### Excel/WPS 中创建趋势图
1. 选中"年度汇总报表"中1-12月的收入和支出列
2. 插入 → 折线图 → 选择样式
3. 双轴展示收入支出月度变化趋势

### Google Sheets 中创建图表
1. 选中数据区域
2. 插入 → 图表
3. 图表类型选择饼图或折线图
4. 右侧面板自定义样式

## 常见问题

**Q: 公式不自动计算？**
A: Excel/WPS中确保公式所在单元格格式为"常规"或"数字"，不要设为"文本"。Google Sheets中公式自动计算。

**Q: 如何添加新的支出分类？**
A: 在"其他"列旁边插入新列，修改合计公式的求和范围即可。

**Q: 月度预算表能否跨年使用？**
A: 可以。复制工作表，将月份改为新一年的即可。模板结构完全复用。

**Q: CSV打开后中文乱码？**
A: 导入时选择UTF-8编码。如已乱码，用记事本打开CSV → 另存为 → 编码选"UTF-8"。
